# 🎨 Creative Portfolio Website

A modern, interactive portfolio website with a sleek dark theme and creative UI/UX designs.

## ✨ Features

- **Dark Theme** - Modern dark background with cyberpunk-inspired gradient colors
- **Responsive Design** - Fully responsive on all devices (mobile, tablet, desktop)
- **Smooth Animations** - Scroll-triggered animations and interactive effects
- **Multiple Sections**:
  - Hero section with call-to-action buttons
  - About me with profile photo and social links
  - Education timeline
  - Projects showcase with GitHub and live demo links
  - Technical skills with category icons
  - Achievements section
  - Contact information
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Hover effects on cards
  - Ripple effect on buttons
  - Counter animations for stats
  - Dynamic role text rotation

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with gradients, animations, and flexbox/grid
- **JavaScript** - Interactivity and smooth animations
- **Font Awesome Icons** - Beautiful icons for skills and social links
- **Placeholders** - Placeholder images for profile and projects

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations

## 🚀 Getting Started

1. **Replace Your Information**:
   - Update "Your Name" in the hero section
   - Replace profile photo URL (currently using placeholder)
   - Update social media links
   - Add your education details
   - Update project information

2. **Customize Colors**:
   - Open `styles.css`
   - Modify CSS variables at the top:
     ```css
     :root {
         --primary-color: #0ff;
         --secondary-color: #ff006e;
         --dark-bg: #0a0e27;
         /* ... more colors ... */
     }
     ```

3. **Add Your Projects**:
   - Replace project images
   - Update project titles and descriptions
   - Add GitHub and live demo links

4. **Update Skills**:
   - Add/remove skill categories
   - Update skill icons (Font Awesome)
   - Modify skill lists

5. **Add Achievements**:
   - Update achievement titles
   - Add your specific achievements
   - Change achievement icons

## 🎨 Customization Tips

### Profile Photo
Replace the placeholder image:
```html
<img src="https://via.placeholder.com/300?text=Your+Photo" alt="Profile Photo">
```
With your own image URL:
```html
<img src="path/to/your/photo.jpg" alt="Profile Photo">
```

### Color Scheme
The portfolio uses cyan (`#00f5ff`) and pink (`#ff0080`) as primary accent colors. Change these in the CSS variables for a different look.

### Adding More Projects
Duplicate the `.project-card` div and update:
- Image URL
- Project title
- Description
- GitHub and demo links
- Technology tags

### Social Links
Update URLs in the social icons:
```html
<a href="https://github.com/your-username" class="social-icon" title="GitHub">
    <i class="fab fa-github"></i>
</a>
```

## 🚀 Deployment

To host this portfolio:

   - Push your files

2. **Netlify**:

3. **Vercel**:
   - Import your repository
   - One-click deployment

4. **Traditional Hosting**:
   - Upload files via FTP
   - Ensure all paths are correct
